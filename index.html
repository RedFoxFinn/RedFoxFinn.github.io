"I think that programming is art."

I might have said it.. And I guess it is true. Even tho it is something that requires some technical and scientific skills, you have to have some eye for structuring the code (which should also work, of course) so it would be readable by others.

Technically, if you write the code just for yourself and never intend to give or show it to others, then you probably could (and might) structure it any way you like it. But in my opinion, good code is always written like you would write it to someone else. You might watch the structure of code yourself after a long time and wonder "What did I think of?" if you don't understand it. (Something you might think even if you did.. but that's completely other thing :)
